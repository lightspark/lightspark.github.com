A free, open source Flash player

Overview

Lightspark is an LGPLv3 licensed Flash player and browser plugin written in C++/C that runs on Linux and Windows. It aims to support all of Adobe's Flash formats.

Current status

Lightspark is still in alpha state, it currently implements around 76% of the Flash APIs. The latest released version is 0.8.3, download it from here.
For up-to-date information regarding support of various websites see this wiki page.
