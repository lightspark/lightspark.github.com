A free, open source Flash player

Overview

Lightspark is an LGPLv3 licensed Flash player and browser plugin written in C++/C that runs on Linux. It aims to support Adobe's newer Flash formats and AVM2 virtual machine.

Current status

The latest released version in 0.8, download it from here.
Lightspark currently supports YouTube well enough for daily use. For up-to-date information regarding support of various other websites see this wiki page.
